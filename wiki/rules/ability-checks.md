# 🎯 Ability Checks & Ability Scores

> Tham khảo: D&D 5e PHB Chapter 7

---

## 6 Ability Scores

| Ability | Viết tắt | Liên quan đến |
|---------|---------|--------------|
| **Strength** | STR | Tấn công melee, athletics, carrying capacity |
| **Dexterity** | DEX | AC (light/no armor), Stealth, Ranged attacks, Finesse |
| **Constitution** | CON | Hit Points, Concentration saves |
| **Intelligence** | INT | Phép thuật Wizard/Eldritch Knight, Arcana, History |
| **Wisdom** | WIS | Perception, Insight, Medicine |
| **Charisma** | CHA | Persuasion, Deception, Intimidation |

---

## Ability Modifier

```
Modifier = (Score - 10) / 2, làm tròn xuống
```

| Score | Modifier |
|-------|---------|
| 1 | -5 |
| 8–9 | -1 |
| 10–11 | +0 |
| 12–13 | +1 |
| 14–15 | +2 |
| 16–17 | +3 |
| 18–19 | +4 |
| 20–21 | +5 |

---

## Ability Check

```
Ability Check = d20 + Ability Modifier (+ Proficiency Bonus nếu Proficient)
```

- So sánh kết quả với **DC (Difficulty Class)**.
- DC thường gặp:

| DC | Độ khó |
|----|--------|
| 5 | Very Easy |
| 10 | Easy |
| 15 | Medium |
| 20 | Hard |
| 25 | Very Hard |
| 30 | Nearly Impossible |

## Passive Checks

```
Passive Score = 10 + Ability Modifier (+ Proficiency nếu có)
```
- Thường dùng cho **Passive Perception** — DM tự check, không cần roll.
- Nếu có Advantage: +5; Disadvantage: -5.

---

## Proficiency Bonus (PB)

| Character Level | Proficiency Bonus |
|----------------|-----------------|
| 1–4 | +2 |
| 5–8 | +3 |
| 9–12 | +4 |
| 13–16 | +5 |
| 17–20 | +6 |

### Expertise
Một số class/feature cho **Expertise** = PB x2 cho skill cụ thể.
